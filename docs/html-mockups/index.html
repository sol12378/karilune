Advertiser

作成元（広告主）

お店・メーカー — 広告を作成し、配信効果を測定する

  • AI支援による低コスト広告作成
  • 下書き → 審査 → 公開のワークフロー
  • 配信者数・参照数・リード数の効果レポート
  • 料金・オプション管理

Distributor

配信者（ガス会社等）

生活インフラ — カタログから選び、会員へ届ける

  • 公開カタログから配信する広告を選択
  • かつて配信した制作元の新作ピックアップ
  • 「お勧め」ピックアップで推薦を明示
  • HP・チラシQR・アプリから会員へ配信
  • 配信実績・会員の反応を確認

Member

消費者（会員）

一般住民 — 顔の見える配信元から情報を受け取る

  • 画像主体の縦フィード（カード型）
  • 「○○ガスからのおすすめ」の信頼シグナル
  • 検索・フィルタなしのパッシブ発見
  • 電話・来店・申込へのリード導線
